CARA INSTAL APLIKASI

1. PASTIKAN SUDAH MENGINSTAL XAMPP PHP 7.1 MINIMAL
2. MEMBUAT DATABASE TERLEBIH DAHULU BISA DI PHPMYADMIN, (UNTUK NAMA DATABASE PASTIKAN SESUAI DENGAN FILE .env YANG ADA DI PROJECT APLIKASI)
3. BUKA COMMAND PROMT KEMUDIAN MASUK FOLDER PROJECT APLIKASI, KEMUDIAN KETIK php artisan migrate (berfungsi mengupload tabel" yang sudah dibuat ke database)
4. KEMUDIAN KETIK php artisan db:seed (untuk mengisi data user admin)
default user admin : 
user : admin@admin.com
password : admin
5. KEMUDIAN KETIK php artisan key:generate
6. JALANKAN SERVER KETIK PERINTAH php artisan serve
